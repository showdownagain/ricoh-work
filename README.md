## 项目简介

理光中国后台管理系统是一个功能完善的企业级管理平台，涵盖了用户管理、智能报备、权限控制、内容发布、商品管理、订单处理等6大核心模块，旨在提升企业运营效率，优化业务流程。

### 🎯 核心价值

- **智能审批引擎**: 根据客户类型和用户角色自动路由审批流程
- **库存协同管理**: 打通经销商库存，实现实时库存确认
- **内容一键导入**: 支持从外部URL自动抓取并导入内容
- **全流程追踪**: 从报备到订单完成的全生命周期管理

## 核心功能

### 1️⃣ 用户管理
- 用户CRUD操作
- 角色分配
- 权限管理
- 批量操作
- 数据导出

### 2️⃣ 智能报备系统 ⭐
- **客户检索**: 全局静默查重
- **客户分类**: 自动判定客户类型（普通/RGS/NAM/央国企/框架）
- **智能路由**: 根据客户类型和角色自动分配审批人
- **多角色支持**:
  - 一级代理商 → 区域渠道支援审批
  - 二级代理商 → 一级代理商初审 → 区域渠道支援终审
  - 直销经理 → 智能路由（市场部担当/区域经理/大客户协调）
- **流程管理**: 冲突申诉、共享申请、状态跟踪

### 3️⃣ 权限管理
- 基于角色的访问控制（RBAC）
- 角色-权限矩阵
- 数据权限隔离
- 9级权限层级

### 4️⃣ 内容管理 ⭐
- **9大内容频道**: 案例库、新闻、产品、解决方案、视频、专题、合作伙伴、活动、通知
- **URL一键导入**: 
  - 输入外部文章URL
  - 自动抓取标题、作者、摘要、正文
  - 一键填充表单
  - 支持案例库和新闻内容
- 富文本编辑
- 图片管理
- 状态控制

### 5️⃣ 商品管理
- 商品CRUD
- SKU管理
- 价格管理
- 分类管理
- 批量导入

### 6️⃣ 订单管理 ⭐
**完整订单生命周期管理**:

**步骤1: 库存确认**
- 客户下单后自动匹配经销商
- 显示经销商联系信息
- 运营人员电话确认库存
- 标注有货/无货
- 有货通知客户付款

**步骤2: 物流管理**
- 客户付款后等待发货
- 经销商发货后填写物流信息
- 物流公司 + 物流单号
- 前台自动通知客户

**步骤3: 确认收货**
- 客户前台确认收货
- 订单状态变为已完成
- 生成完整报表

**步骤4: 报表统计**
- 完整订单明细
- 一键导出CSV
- 包含所有关键字段

## 功能特色

### 🚀 智能报备系统

#### 客户类型自动识别
```typescript
客户类型体系:
├── 普通客户     → 标准审批流程
├── RGS客户      → 区域经理审批
├── NAM客户      → 全国大客户专审
├── 央国企大客户  → 大客户协调专审
└── 框架客户     → 区域渠道支援审批
```

#### 智能审批路由
```typescript
角色 + 客户类型 → 自动分配审批人

一级代理商:
  普通客户 → 区域渠道支援

二级代理商:
  任何客户 → 一级代理商初审 → 区域渠道支援终审

直销经理:
  普通客户 → 市场部担当（姚俊）
  RGS/NAM → 区域经理 → (拒绝可上诉)直销副总经理
  央国企 → 大客户报备协调（王凡/连紫昭）
```

#### 全流程追踪
- ✅ 提交报备
- ✅ 客户查重
- ✅ 类型判定
- ✅ 路由分配
- ✅ 审批流转
- ✅ 状态更新
- ✅ 通知推送
